1. Отидете до папката server и стартирайте локалния сървър. (по ваш установен и удобен начин)

Например чрез терминала може да използвате следните команди:
cd server (ако се намирате 1 папката нагоре от server папката, ще ви заведе в нея)
node ./server.js  (вече ще ви стартира сървъра)

2. Отворете конкретната задача в VS Code и я стартирайте, като използвате live server/lite-server или по какъвто
начин ви е удобно и желаете.

Всички задачи са съобразени спрямо изискванията на условията. На някои задачи липсва добро обяснение за някои крайни
случаи, а на други информацията която се получава от базата на сървара се разминава с това по условието.

Надявам се, че след като разцъкате и разгледате кода да установите, че съвпада с изискванията по задачите.
Моля за обратна връзка и вашите коментари!
